Não me lembro da sintaxe do if-else...

Então lá vai um exemplo: 

if temperatura > 30:
    print("Está calor")
elif temperatura < 15:
    print("Está frio")
else:
    print("Está agradável")

Meu código está ficando muito grande, tem como eu diminuí-lo?

Sim, podemos usar os operadores lógicos para utilizar mais de uma expressão booleana. São eles:

  • and - retorna verdadeiro apenas se as duas expressões forem verdadeiras
  • or - retorna verdadeiro se pelo menos uma das expressões for verdadeira
  • not - inverte o valor booleano, se for True vira False e vice versa

E como posso usá-lo?

Vou dar um exemplo do uso do and: 

if temperatura < 30 and temperatura > 15:
    print("Está agradável")

Não consigo escrever a frase solicitada.

Para escrever a frase inteira como texto, precisamos transformar o valor inteiro em texto. Para isso, pegue a idade do irmão mais velho e use o comando str() como indicado abaixo: 

str(idade_maior)

E como adiciono isso à frase completa?

Usamos o sinal de + para concatenar (juntar) pedaços de uma frase. O código ficaria assim: 

"O irmão mais velho tem " + str(a) + " anos"

Não se esqueça de colocar espaço dentro das aspas, para que a idade não fique colada à frase.
