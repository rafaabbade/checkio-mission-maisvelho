Enunciado

Sofia, planejadora de festas de aniversário, frequentemente organiza festas para famílias grandes. Ela quer adicionar um toque personalizado às festas, destacando o irmão mais velho em famílias com três crianças. Sofia precisa de uma função que identifique rapidamente qual dos irmãos é o mais velho.

Entrada: Três números inteiros diferentes representando as idades dos irmãos.

Saída: Uma string contendo a mensagem "O irmão mais velho tem X anos", onde X é a idade do irmão mais velho.
